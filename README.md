# ToolsChallenge

Esse projeto é uma API de pagamentos, utilizando os conceitos REST, JSON e protocolo HTTP de retorno, os padrões MVC e DTO.

# Tecnologias utilizadas
- Java
- Spring Boot
- Spring MVC
- JPA / Hibernate
- Maven
- JUnit5
- Mockito
